HTML (HyperText Markup Language)

HTML adalah bahasa markup standar yang digunakan untuk membuat dan merancang halaman web. Dengan HTML, Anda dapat menentukan struktur dasar sebuah halaman web, termasuk judul, paragraf, daftar, gambar, tautan, dan banyak lagi.

CSS (Cascading Style Sheets)

CSS adalah bahasa gaya yang digunakan untuk mengatur tampilan dan tata letak elemen-elemen HTML di halaman web. Dengan CSS, Anda dapat mengontrol warna, font, ukuran, jarak, dan berbagai properti gaya lainnya dari elemen-elemen HTML.

JavaScript

JavaScript adalah bahasa pemrograman yang sering digunakan untuk membuat situs web interaktif dan dinamis. JavaScript berjalan di sisi klien (di browser) dan memungkinkan interaksi pengguna, manipulasi DOM, validasi formulir, animasi, dan banyak lagi.
